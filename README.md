# DUSKMOURN · DSK

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **ARENA**.

DUSKMOURN: HOUSE OF HORROR (DSK) — the first sphere of the ARENA Magic: The Gathering domain: the WHOLE set curated by basic worth. Pulls all ~278 cards LIVE from Scryfall (card data + the non-foil USD it mirrors from TCGplayer) and buckets them into worth tiers — 🩸 chase ($5+) · staples ($1–5) · role-players ($0.25–1) · bulk (&lt;$0.25) — with a set-value summary (sum, top card, chase count), search, rarity filter, and a foil-worth toggle; each card shows its rarity, type, and basic worth beside a Scryfall thumbnail. The curation is the tiering + the read applied to REAL numbers, not invented ones (verified live: 278 cards, ~$286 non-foil set value, top Overlord of the Balemurk ~$21.56, 18 chase cards). Honest: prices are a live snapshot and fluctuate daily, an unlisted card shows —; card names &amp; prices are facts, images are hot-linked from Scryfall's CDN (sanctioned non-commercial use); an unaffiliated fan price-reference — Magic and the card names / art are © Wizards of the Coast. Built to test the domain: give a set, get a sphere. David Lee Wise / Bridge-Burners LLC.
